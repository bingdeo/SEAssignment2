### 회원 가입
| Actor Action                 | System Response     |
| ---------------------------- | ------------------- |
|1. 회원이 ID, 비밀번호, 전화번호 입력|2. 입력한 ID, 비밀번호, 전화번호 출력|

### 로그인
| Actor Action                 | System Response     |
| ---------------------------- | ------------------- |
|1. 아이디와 비밀번호 입력|2. ID, 비밀번호 출력|

### 로그아웃
| Actor Action                 | System Response     |
| ---------------------------- | ------------------- |
|1. 사용자가 로그아웃 요청|2. ID 출력|

### 자전거 정보 등록

### 자전거 대여
| Actor Action                 | System Response     |
| ---------------------------- | ------------------- |

### 자전거 대여 정보 조회
| Actor Action                 | System Response     |
| ---------------------------- | ------------------- |
|1. 회원은 대여 중인 자전거 조회 요청|2. 자전거 ID, 자전거 제품명 리스트 출력|
